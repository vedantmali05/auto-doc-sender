# Auto Doc Sender

Auto Doc Sender is a reusable Python-based automation tool that helps you:

- Add participant names to PDF documents (certificates, letters, etc.)
- Send those personalized PDFs automatically via email
- Reuse the same system for multiple workshops or events

This project is designed to be simple, flexible, and beginner-friendly.

---

## Features

- Automatic PDF personalization
- Bulk email sending
- Uses Excel files for participant data
- Reusable for any event
- No manual work after setup

---

## Use Cases

- Workshop certificates
- Event participation certificates
- Offer letters
- Any personalized PDF document

---

## Basic Workflow

1. Prepare a PDF template with blank space for the name
2. Prepare an Excel file with names and email addresses
3. Run the Python script
4. Documents are generated and emailed automatically

---

## Requirements

- Python 3.8 or higher
- Internet connection (for sending emails)
